## Objective
Understand what a home lab is and why it is used in cybersecurity.

## What is a Home Lab?
A home lab is a personal practice environment used to learn, test, and experiment with technologies such as operating systems, networking, and security tools without affecting real systems.

## Why Home Labs Are Important
- Hands-on learning
- Safe environment for testing
- Builds real-world skills
- Helps understand system behavior

## Tools Used
- Personal computer
- Virtualization software (planned)
